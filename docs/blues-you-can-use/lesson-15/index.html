Practice​

  • Play scales using string pairs:
    • Start with the lowest complete pattern available in a given key.
    • Play the notes of the scale on the 6th and 5th string, from the lower note on string 6 to the upper note on string 5.
    • Move up to the next pattern and play the notes in the same way.
    • When you reach the top pattern playable, reverse the process: now play the upper note of the 5th string down to the lower note of the 6th string, moving all the way down the neck.
    • Repeat this procedure on the rest of the string pairs (5th and 4th, 4th and 3rd, and so on).
    • Shift from one pattern to the next as smoothly as possible so that you can't hear the shift.
